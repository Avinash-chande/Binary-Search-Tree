# Binary Search Tree (BST)

[![C++](https://img.shields.io/badge/language-C++-blue)]()  


A repository for implementations of **binary trees** and **binary search trees** (BST), supporting various operations and traversal techniques.

**Repository URL:**  
https://github.com/Avinash-chande/Binary-Search-Tree

---


## Overview

This project contains clean implementations of binary trees and binary search trees in C++. The goal is to serve both as:

- A learning resource for students and developers,  
- A foundation that you can extend (e.g. to self-balancing BSTs).

It includes:

- Insert, search, delete operations in BST  
- Traversals (inorder, preorder, postorder)  
- Both recursive and iterative traversal techniques  
- Modular and extensible code structure  

---

## Features

- Simple, readable C++ code  
- Both recursive and iterative traversal methods  
- Standard BST operations  
- Easily extendable to advanced BST types (AVL, Red-Black, etc.)  
- Useful for algorithm practice, learning, and interviews
